🎬 I DO — Analysis & Vocabulary

Why do movie companies create posters?

Movie posters serve multiple purposes — they attract audiences, set the tone/genre, and act as brand identity. Every design choice is deliberate:

Anatomy of a Movie Poster

Main Image / Hero Shot
Establishes character and tone. Often the lead actor or key visual symbol.
Title / Typography
Font style signals genre — scratchy for horror, bold for action, elegant for drama.
Tagline
A short, memorable phrase that hooks the audience. Creates mystery or excitement.
Colour Palette
Dark blues/greens = thriller. Bright/warm = comedy. Washed out = dystopia.
Billing Block
Small text at the bottom — actors, director, production company, rating.
Studio Logos
Warner Bros, Universal, A24 etc. — signals production quality and budget.

Target Audience Analysis

Designers consider age, interests, gender, and viewing habits when designing a poster. Compare:

FLASH GORDON
  • Dark, intense colours
  • Muscular, dramatic figures
  • Target: adult sci-fi fans, nostalgia
COCO (PIXAR)
  • Vibrant, warm colours
  • Cartoon style, friendly characters
  • Target: families, children

🎬 I DO — Photopea Tour
▶ OPEN PHOTOPEA

Key Tools in Photopea

1

Opening the Template

File → Open → select your PSD template. If you don't have it: File → New → set width 595px, height 842px (A4 portrait)

2

Understanding Layers

The Layers panel (right side) shows every element stacked. Higher layers sit on top. Use it to reorder, hide, or lock layers.

3

Adding a Background

File → Place → find your background image. Drag the corners to resize. Press Enter to confirm. Move this layer to the BOTTOM of the layers panel.

4

Placing Character Images

File → Place your character. Resize using the corner handles (hold Shift to keep proportions). Press Enter to confirm.

5

Removing Backgrounds

Select your character layer → in the top menu: Image → Remove Background (AI magic!) OR use the Magic Wand tool to click the background, then press Delete.

💡 LAYERS — THE MOST IMPORTANT THING

Think of layers like sheets of transparent paper stacked on top of each other. Your background goes on the bottom sheet, characters go in the middle, and text goes on top. Never work on the wrong layer!

  • Or: Search Google Images for "your subject PNG transparent" to find images with backgrounds already removed
🌟 EXTENSION
  • Add a gradient overlay layer above your background (create a new layer, use the Gradient tool, set layer blend mode to "Multiply" or "Screen")
  • Add a Curves adjustment layer to create a cinematic colour grade

🎬 I DO — Text & Effects
1

Adding Text

Select the Type Tool (T) from the toolbar. Click on your canvas and type. Use the top bar to change font, size, and colour. Put each text element on its own layer.

2

Layer Styles / Effects

Double-click a layer → Layer Style opens. Add: Drop Shadow, Glow, Stroke (outline), Bevel. These add depth and drama to your text and images.

3

Blend Modes

At the top of the Layers panel, change "Normal" to modes like Multiply, Screen, Overlay. This creates dramatic lighting effects and atmosphere.

4

Filters

Filter menu → blur, sharpen, distort your images. Use sparingly — a little goes a long way!

5

Exporting

File → Export As → JPEG (for sharing) OR keep the PSD for editing later. Save JPEG at 90%+ quality.
