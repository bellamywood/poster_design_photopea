"A portal has opened. Not everyone will survive."
In 3025, Earth's greatest minds face catastrophe when a mysterious rift tears open in deep space, unleashing an unstoppable alien force. You have been commissioned to design the official theatrical movie poster. It must feel vast, dangerous, and cinematic — a world on the edge of destruction.

STEP 03

BUILD YOUR POSTER IN PHOTOPEA

🎯 Key Objectives

ALLOpen Photopea, set up the canvas, and place your background and hero image
MOSTLayer multiple images, remove backgrounds, and adjust sizes to create depth
SOMEUse blend modes, colour adjustments, and effects to create a cinematic, professional result
▶ OPEN PHOTOPEA
1

Set Up Your Canvas

File → New → click the Print tab → select A4. Then type galactic_rift_poster as the name. Click Create.

2

Place Your Background

File → Place → select your space background. Drag the corners to fill the entire canvas. Press Enter to confirm. In the Layers panel, drag this layer to the very bottom.

▶ Watch first — How to set up Photopea and place your background

3

Add Your Hero Character

File → Place → select your astronaut or figure. Resize using corner handles (hold Shift to keep proportions). Press Enter. This layer goes above the background.

4

Remove the Background from Your Character

Select the character layer → Image → Remove Background. Or use the Magic Wand tool: click the background area → press Delete. Clean up any remaining edges.

▶ Watch first — How to remove the background from your hero image

5

Add Props & Create Depth

Place your spacecraft, rift, or objects. Size them so that objects closer to the viewer appear larger — this creates perspective. Reorder layers in the Layers panel to control what appears in front.

6

Save Your Progress

File → Save As → choose Photopea PSD format. Wait for the white saving bar to fully disappear before doing anything else. Save regularly!

▶ Watch first — How to check your work and save

Build Checklist

🪜 SUPPORT — CAN'T REMOVE THE BACKGROUND?
  • Try: Image → Remove Background (works best with clear contrast between subject and background)
  • Or: Select the Magic Wand tool → click the background → press Delete
  • Or: Look in the Movie Poster Assets folder for images that already have a plain or solid-colour background — these are much easier to remove
🌟 EXTENSION — CINEMATIC TECHNIQUES
  • Add a gradient overlay: create a new layer, use the Gradient tool (dark at edges, transparent in centre), set blend mode to "Multiply". This creates a vignette — used in almost every sci-fi poster.
  • Add a Curves adjustment layer (Layer → New Adjustment Layer → Curves) and pull the blue channel up slightly. This is how films achieve a cold, futuristic colour grade.
  • Duplicate your background layer, apply Filter → Blur → Gaussian Blur, then reduce the opacity to 40%. This adds an atmospheric haze to the scene.

⚠ WARNING: Save your PSD file BEFORE closing Photopea. Wait for the white saving bar to disappear completely!

Build Checklist

🪜 SUPPORT — CAN'T REMOVE THE BACKGROUND?
  • Try: Image → Remove Background (works best with clear contrast between subject and background)
  • Or: Select the Magic Wand tool → click the background area → press Delete
  • Or: Look in the Movie Poster Assets folder for images that already have a plain or solid-colour background — these are much easier to remove
🌟 EXTENSION — BRING THE ENERGY
  • Add a warm gradient overlay: create a new layer, use the Gradient tool (yellow-orange at the top, transparent at the bottom), set blend mode to "Screen" at 30% opacity. This gives the scene a warm, sunny, cinematic glow.
  • Try rotating one of the animals slightly (using the Transform tool) to make the composition feel more chaotic and fun — like everyone's running in different directions.
  • Add a drop shadow to each animal layer (double-click the layer → Layer Style → Drop Shadow). This grounds the animals in the scene and makes them feel like they belong in the environment.

⚠ WARNING: Save your PSD file BEFORE closing Photopea. Wait for the white saving bar to disappear completely!
