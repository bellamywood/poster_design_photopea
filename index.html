GALACTIC RIFT
"A portal has opened. Not everyone will survive."

Dark & atmospheric Cold cosmos Glowing energy Sense of scale

STEP 02 WHAT DO YOU NEED TO FIND?

Create a folder called assets in your Year 8 Computing folder. You need to find and save these image types for GALACTIC RIFT:

🌌
Background
A dramatic space or deep-sky scene that fills the canvas
→ Nebula or galaxy photograph → Star field with colour → Planet with atmosphere glow
🧑‍🚀
Hero Character
A human figure — astronaut, scientist, or soldier
→ Astronaut in suit (full body) → Silhouette of a person → Dramatic portrait / close-up
🛸
Objects / Props
Sci-fi objects that set the scene
→ Spaceship or spacecraft → Portal / energy rift PNG → Alien structure or technology
🎬
Studio Logo
A real studio logo as a transparent PNG
→ Warner Bros, Universal → Paramount, A24 → 20th Century Studios

STEP 03 BUILD YOUR POSTER IN PHOTOPEA

🎯 Key Objectives

ALLOpen Photopea, set up the canvas, and place your background and hero image
MOSTLayer multiple images, remove backgrounds, and adjust sizes to create depth
SOMEUse blend modes, colour adjustments, and effects to create a cinematic, professional result

Open Photopea before moving on — it runs in your browser, no install needed. Keep this workbook open in a separate tab so you can follow along step by step.

▶ OPEN PHOTOPEA
STEP 03 SET UP YOUR CANVAS
1

Set Up Your Canvas

File → New → click the Print tab → select A4. Type galactic_rift_poster as the name. Click Create.

2

Place Your Background

File → Place → select your space background. Drag the corners to fill the entire canvas. Press Enter to confirm. In the Layers panel, drag this layer to the very bottom.

▶ Watch — How to set up Photopea and place your background

STEP 03 ADD YOUR HERO CHARACTER
3

Add Your Hero Character

File → Place → select your astronaut or figure. Resize using corner handles (hold Shift). Press Enter. This layer goes above the background.

4

Remove the Background from Your Character

Select the character layer → Image → Remove Background. Or use the Magic Wand tool: click the background area → press Delete. Clean up any remaining edges.

▶ Watch — How to remove the background from your hero image

🪜 CAN'T REMOVE THE BACKGROUND?
  • Try: Image → Remove Background (works best with clear contrast)
  • Or: Select the Magic Wand tool → click background area → press Delete
  • Or: Use images from the assets folder that already have a plain background
STEP 03 ADD DEPTH & SAVE
5

Add Props & Create Depth

Place your spacecraft, rift, or objects. Size them so objects closer to the viewer appear larger — this creates perspective. Reorder layers in the Layers panel to control what's in front.

6

Save Your Progress

File → Save As → choose Photopea PSD format. Wait for the white saving bar to fully disappear before doing anything else. Save regularly!

▶ Watch — How to check your work and save

STEP 03 BUILD CHECKLIST
🌟 EXTENSION — CINEMATIC TECHNIQUES
  • Add a vignette: new layer, Gradient tool (dark at edges, transparent centre), blend mode "Multiply".
  • Add a Curves adjustment layer and pull the blue channel up slightly for a cold, futuristic colour grade.
  • Duplicate your background, apply Filter → Blur → Gaussian Blur, reduce opacity to 40% for atmospheric haze.

⚠ SAVE your PSD file BEFORE closing Photopea. Wait for the white saving bar to disappear completely!
